# Audyt UX i funkcjonalny aplikacji edukacyjnej - Duolingo (iOS, darmowa)

## Cel projektu
Celem projektu jest wykonanie kompleksowego audytu UX i funkcjonalnego aplikacji edukacyjnej dla młodzieży w wieku 13-15 lat. Analiza obejmuje:
- onboarding (tworzenie konta, konfiguracja języka, wybór poziomu i celu nauki),
- przebieg jednej lekcji (flow, feedback, elementy interfejsu),
- heurystyki UX (wg Nielsena),
- dostępność i bezpieczeństwo poznawcze,
- analizę ryzyk i rekomendacje dla użytkowników i zespołu produktowego.

---

## Grupa docelowa
- Wiek: 13-15 lat
- Platforma: iOS
- Wersja aplikacji: darmowa

---

## Metodologia
- Audyt przeprowadzono zgodnie z zasadami UX i QA: obserwacja flow, analiza komunikatów, testowanie funkcjonalności.  
- Uwzględniono heurystyki Nielsena i dobre praktyki edukacyjne dla młodzieży.  
- Testy obejmowały scenariusze z błędnymi i poprawnymi odpowiedziami oraz korzystanie z aplikacji offline.  

---

## Wnioski ogólne
- Flow aplikacji jest intuicyjny i wspiera motywację użytkownika.  
- Obszary do poprawy: tłumaczenie błędów, informacje o maksymalnej liczbie punktów, natarczywe powiadomienia, przepływ widgetu w onboarding.  
- Pozytywne elementy: czytelny interfejs, responsywność, motywujący feedback, możliwość korzystania bez dźwięku, stonowana kolorystyka.
